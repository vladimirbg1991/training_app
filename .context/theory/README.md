# Fitness Theory Reference Library

This folder is for fitness science, training methodology, and exercise reference material that the fitness-domain-expert agent draws on.

**Files to add later (not blocking the project bootstrap):**

- `core-50-exercises.md` — the seed catalog of 50 exercises with full domain shape (name, primary muscle, secondary muscles, equipment, category, instructions). Generate this with the fitness-domain-expert during Phase 0.
- `progressive-overload-models.md` — the 1RM formulas (Epley, Brzycki), volume math, and the clamping rules that prevent algorithmic hallucinations.
- `equipment-taxonomy.md` — the precise equipment categorization (leverage_machine vs selectorized_machine vs cable) that distinguishes our catalog from competitors.
- `training-methodology-glossary.md` — RPE, RIR, periodization, deload, and other terms used in the UI and analytics.

Files in this folder are read-only references; no agent writes here.
